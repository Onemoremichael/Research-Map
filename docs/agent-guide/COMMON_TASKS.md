# Common Agent Tasks

<!-- reviewed: 2026-02-24 -->

## Start a New Research Task

1. Check `plans/active/` for related plan.
2. If needed, create a plan from `plans/_TEMPLATE.md`.
3. Define objective, hypothesis, metrics, and artifacts.
4. Initialize templates under `docs/research-ops/` for intake/hypotheses/experiments.
5. Execute scoped implementation/experiments.
6. Document evidence and update docs.

## Add or Update Paper Content

1. Follow `docs/workflows/PAPER_WRITING.md`.
2. Map each claim to evidence.
3. Verify citations from trusted sources.
4. Mark unresolved citations as placeholders.
5. Update docs and handoff.

## Update Documentation

1. Edit target doc.
2. Update freshness tag.
3. Validate links, structure, and docs index coverage.

## End Session

1. Update `docs/session/SESSION_HANDOFF.md`.
2. Capture completed work, in-progress items, blockers, and next steps.

# Architecture

> Detailed documentation lives in `docs/architecture/`.

## Summary

Research_Map uses progressive disclosure:

```
Agent Entry Files
    -> docs/_INDEX.md
        -> specific operational docs
            -> code, experiments, and artifacts
```

## Research-Oriented Principles

1. One truth in `docs/`
2. Plans track temporal work in `plans/`
3. Evidence and claims must stay aligned
4. Paper-writing and citation integrity are first-class workflows
